Evaluador de expresiones en Python

Este programa es una herramienta que evalúa expresiones, como por ejemplo "2+2*3+5*a". Comienza solicitando el valor de 'a' y luego resuelve paso a paso, mostrando el tipo de datos de cada número involucrado.

Además, el software tiene la capacidad de resolver expresiones contenidas en un archivo de texto. En el archivo, cada expresión debe estar en una línea separada, por ejemplo:

                    4+5*a
                    7+8*10

También puede resolver expresiones individuales ingresadas directamente, como por ejemplo "4*a+99".
